Comment Jouer:

1. lancer Programme java

2. 	But du Jeux: Arriver jusqu'à drapeaux d'arriver en évitant tous
	les obstacles pour ne pas mourir et récolter un maximum de points

3. Contrôles:
	Flèche du haut : accélérer 
	Flèche du bas : freiner 
	Flèche de Droite : Rotation vers la Droite 
	Flèche du Gauche : Rotation vers la gauche 
	Touche Espace : Pivoter le personnage de 180° 
	Touches R : Recommencer 
	Touche Shift : Sauter
